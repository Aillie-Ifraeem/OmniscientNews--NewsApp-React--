📰 Omniscient News App (React)

A real-time news application built with React that fetches latest news using NewsAPI and displays it in a clean UI with infinite scrolling.

---

🚀 Features
- Live news updates using NewsAPI
- Category-based news display
- Infinite scrolling
- Loading spinner for better UX
- Responsive UI

---

🛠️ Tech Stack
- React.js
- JavaScript (ES6+)
- CSS
- NewsAPI
- Bootstrap (if used)

---

 🔑 API Setup
This project uses NewsAPI.

Create a `.env` file in root:
REACT_APP_NEWS_API_KEY=your_api_key_here

⚠️ Never expose your API key publicly.
